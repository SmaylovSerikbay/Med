## Быстрый старт

```bash
# Вариант 1: Прямая команда Django
python manage.py generate_test_data

# Вариант 2: Через bat-файл (Windows)
generate_test_data.bat

# Вариант 3: Через Docker
docker-compose exec backend python manage.py generate_test_data
```
